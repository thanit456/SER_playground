# SER_playground

## Dataset : (IEMOCAP)
- 10 unique speakers 
- 6 labels (happy, excited, neutral, sadness, disgust, angry, frustrated)

## Model
(1.) TRE model (only transcription input)
(2.) ARE model (only audio input)
(3.) MDRE model (audio and transcription)
(4.) MDREA model (audio and transcription)

## Prediction
- 4 emotions (happy, neutral, sadness, angry)
